## Task Tracker

A simple vanilla HTML/JS task tracker for practicing agentic coding workflows.

### Run locally
```bash
# Simply open index.html in a browser, or use a simple server:
npx serve .
# or
python -m http.server 8000
```

### Features
- Add/delete tasks
- Mark tasks complete
- Filter: All / Active / Completed
- LocalStorage persistence
- Clear completed
